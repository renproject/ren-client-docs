Contract: LinkedList#

LinkedList is a library for a circular double linked list.

insertBefore (struct LinkedList.List self, address target, address newNode) (internal)#

Params

  • self: The list being used.

  • target: The existing node in the list.

  • newNode: The next node to insert before the target.

Insert a new node before an existing node.

remove (struct LinkedList.List self, address node) (internal)#

Params

  • self: The list being using.

  • node: The node in the list to be removed.

Remove a node from the list, and fix the previous and next pointers that are pointing to the removed node. Removing anode that is not in the list will do nothing.
