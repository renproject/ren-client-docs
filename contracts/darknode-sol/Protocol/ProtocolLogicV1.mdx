## Contract: `ProtocolLogicV1`

ProtocolLogicV1 implements the getter functions for the Protocol proxy
as well as onlyOwner functions for updating the values in ProtocolState.



Inherits from:

- `IGatewayRegistry`
- `ProtocolStateV1`
- `Claimable`
- `Ownable`
- `Context`
- `Initializable`
