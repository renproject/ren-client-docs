Tokenized Asset logos#

Once an asset is in the custody of RenVM, a token at a 1:1 value is minted on the destination chain - this can then be used as any token can within that destination chain's ecosystem (for example, swaps, collateralization, trades etc.). To visually differentiate between the underlying asset (for example, BTC) and its tokenized counterpart, we‚Äôve developed a set of ‚ÄúTokenized Asset‚Äù logos.

We've tried as much as possible to create a seamless user experience where the end-user is not even aware of the Tokenized Asset - where possible we‚Äôve tried to ensure the end-user feels like they‚Äôre interacting with the underlying asset.. However, there are some instances where you‚Äôll need to use the Tokenized Asset logo, such as blockexplorers.

If you‚Äôre unsure whether to use the underlying asset‚Äôs logo or the tokenized asset logo, get in touch with our team in the RenVM Developer Center Telegram group.
